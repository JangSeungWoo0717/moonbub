# 🎯 포물선 슈터 (ParabolaShot)

발사대(원점)에서 공이 **이차함수 `y = a·x² + b·x`** 궤적을 그리며 날아가는 퍼즐
아케이드 게임. 계수 a·b를 조절해 표적을 맞히고 장애물을 피한다.

- 과목/주제: 수학 · 「이차함수로 조준하라 — 포물선의 기하」 (고1)
- 목표 학과: 게임학과 (학생부종합전형 세특)

## 핵심 기능
- a·b 슬라이더로 포물선 **궤적 실시간 미리보기**
- 발사 시 공이 궤적을 따라 애니메이션
- 표적 명중 / 장애물 충돌 판정, 5단계 레벨
- 시도 효율에 따른 **별점(1~3)** 과 진행도 저장(localStorage)
- 현재 포물선의 꼭짓점·대칭축·착지점(x절편) 표시

## 수학 개념 (고1 이차함수)
- 궤적: `y = a·x² + b·x` (상수항 0 → 항상 발사대에서 출발)
- a의 부호·크기 → 개구 방향과 폭
- 꼭짓점·대칭축: `x = -b/2a`
- 착지점(x절편): `x = -b/a`

## 세특 서사
이차함수의 계수가 포물선 모양·꼭짓점·x절편을 어떻게 바꾸는지 **게임으로 탐구**했다.
발사 물리, 충돌 판정, 레벨 디자인, 별점 보상을 직접 설계해 수학을 게임의 조준
메커니즘으로 사용했다. (수학적 이해 + 게임 시스템 설계 역량 증빙)

## 실행
정적 사이트입니다. 로컬에서:

```
python -m http.server 8000
```

브라우저에서 `http://localhost:8000/수학/` 접속.

## 테스트

```
npm test   # node --test (포물선 수학·충돌·게임 상태·레벨 풀이 가능성)
```

순수 로직(physics/levels/game/storage)은 DOM·Canvas 비의존으로 단위 테스트된다.

## 기술
HTML/CSS/순수 ES Modules, Canvas 2D, localStorage. 빌드 없음.

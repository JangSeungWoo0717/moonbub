# 🎯 포물선 슈터 (ParabolaShot)

대포를 드래그해 조준하면 공이 **이차함수 `y = a·x² + b·x`** 궤적으로 날아가는
퍼즐 아케이드 게임. 하나의 포물선으로 **여러 표적과 코인을 동시에** 꿰어야 한다.

- 과목/주제: 수학 · 「이차함수로 조준하라 — 포물선의 기하」 (고1)
- 목표 학과: 게임학과 (학생부종합전형 세특)

## 핵심 기능
- **드래그 조준**(파워+각도 → 계수 a·b) + 슬라이더 미세 조정
- **궤적 실시간 미리보기**(꼭짓점·대칭축·착지점 마커)
- **다중 표적 + 코인 수집**: 곡선 하나로 모두 통과해야 하는 정밀 퍼즐
- 벽·천장 장애물(막히면 공이 벽에서 멈춤)
- **12개 레벨**, par·코인·명중 조합 **별점(1~3)**, 진행도 저장
- 손맛 연출: 파티클 폭발, 공 잔상·발광, 화면 흔들림, 풍선 표적, **WebAudio 효과음**(에셋 없음)

## 수학 개념 (고1 이차함수 × 물리)
- 발사 속도 (vₓ, v_y)와 중력 g → 궤적 `y = -(g/2vₓ²)x² + (v_y/vₓ)x`
- a = -g/2vₓ² (항상 아래로 열림, 빠를수록 완만)
- b = v_y/vₓ (발사 기울기·각도)
- 꼭짓점 `x = -b/2a`, 착지점(x절편) `x = -b/a`

## 세특 서사
물리(발사)와 이차함수의 연결을 게임으로 탐구했다. 여러 표적·코인이 하나의 포물선
위에 놓이도록 계수를 조절하는 퍼즐을 설계하고, 충돌 판정·레벨 디자인·별점 보상·
파티클/사운드 연출을 직접 구현했다. (수학적 모델링 + 게임 시스템 설계 역량 증빙)

## 실행
정적 사이트입니다. 로컬에서:

```
python -m http.server 8000
```

브라우저에서 `http://localhost:8000/수학/` 접속.

## 테스트

```
npm test   # node --test
```

순수 로직(physics·scoring·levels·game·storage)은 DOM/Canvas 비의존으로 단위
테스트되며, **모든 레벨이 조작 범위 안에서 표적+코인 완전 클리어가 가능함**을
전수 탐색으로 검증한다.

## 기술
HTML/CSS/순수 ES Modules, Canvas 2D, WebAudio, localStorage. 빌드 없음.

# 🎲 확률의 설계자 (GachaLab)

게임의 뽑기(가챠) 시스템을 **확률·기댓값·시뮬레이션**으로 분석하는 확률 실험실.

- 과목/주제: 수학 · 「확률로 설계하는 게임 — 기댓값과 의사결정」 (고1)
- 목표 학과: 게임학과 (학생부종합전형 세특)

## 핵심 기능
- 레어도·천장(pity) 설정 → 목표 획득 **기대 뽑기 수** 계산
- **누적확률 곡선** (N번 안에 1개 이상 얻을 확률, 여사건)
- 50% / 90% / 99% 달성에 필요한 뽑기 수
- **몬테카를로 시뮬레이션** → 분포(히스토그램)·평균±표준편차, 이론값과 비교(큰 수의 법칙)
- 천장 적용 기댓값 비교, 단발 뽑기 연출

## 수학 개념
- 여사건: `1 - (1-p)^N`
- 기댓값(기하분포): `1/p`
- 분위 뽑기 수: `⌈ln(1-q) / ln(1-p)⌉`
- 천장 기댓값: `Σ k·(1-p)^(k-1)·p + H·(1-p)^(H-1)`
- 몬테카를로 + 큰 수의 법칙

## 세특 서사
확률을 게임 밸런싱·공정성 설계의 도구로 다뤘다. 여사건·기댓값으로 가챠를 분석하고,
천장 설계가 기댓값과 최악의 경우를 어떻게 바꾸는지 실험했으며, 몬테카를로로 큰 수의
법칙을 확인했다. (수학적 모델링 + 게임 설계 역량 증빙)
